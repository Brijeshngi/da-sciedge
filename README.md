
# SciEdge Innovations - React + MUI (Watery Glass Theme)

Fully responsive landing app using **Vite + React + MUI** with a watery glass aesthetic, theme toggle, GA4 hooks, and EmailJS contact form.

## Env Vars (create `.env`)
```
VITE_GA_MEASUREMENT_ID=G-XXXXXXX
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## Run
```
npm install
npm run dev
```

This build is English-only (no i18n). The About and Contact sections are customized per your instructions.
